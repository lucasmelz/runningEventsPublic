About the company

This is a mockup website of a company that promotes and helps to organize running events, providing a web solution for event organizers. The website allows athletes to register and enroll for running, also maintaining a persistent database of their classifications and other statistics. Administrators can register new events and update the data concerning the athletes' performance in each race. Developed by Lucas Werle Melz (44224).
