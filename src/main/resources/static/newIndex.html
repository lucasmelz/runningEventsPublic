About the company.

This is a mockup website of a company that promotes and helps to organize running events, providing a web solution for event organizers. The website allows athletes to register and enroll for running, also maintaining a persistent database of their classifications and other statistics. Administrators can register new events and update the data concerning the athletes' performance in each race. This platform is a RESTful Webapp implemented with the SpringBoot framework, making use of JPA and a PostgreSQL database. It was developed by Lucas Werle Melz.
